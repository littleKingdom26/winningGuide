# 🎯 필승 가이드 - 추가 기능 제안

## 📊 현재 상태 분석

### ✅ 구현된 기능
1. **메인 대시보드** - 검색, 카테고리, HOT 응원법
2. **상황별 응원** - GOAL, GAMEOVER, EXTRATIME, GENERAL 등
3. **선수별 응원** - 선수 프로필 & 응원가
4. **검색** - 선수명, 응원가 검색
5. **관리자 페이지** - 응원가/선수 CRUD
6. **API** - 데이터 백업, 플레이어, 응원가 관리

### 📁 현재 데이터 구조
- **CheerSong**: id, title, category, lyrics, videoUrl, targetPlayerId, tags, createdAt, notes
- **PlayerProfile**: id, backNumber, name, position, thumbnailUrl, cheerSongId

## 🚀 우선순위별 추가 기능 제안

### 🔥 P0 (즉시 추천) - 실용성 최고

#### 1. ⚽ 경기 일정 & 실시간 정보
```
기능: 다가오는 경기 일정, 실시간 스코어, 경기장 안내

추가 데이터:
- GameSchedule: id, opponent, home/away, date, time, venue, weather, status
- LiveMatch: id, homeScore, awayScore, currentTime, events[]

UI 위치:
- 메인 페이지 상단 "다가오는 경기" 카드
- "경기장 안내" 전용 페이지

이유: 직관 경객들이 가장 먼저 찾는 정보!
```

#### 2. 🔔 경기 알림/푸시
```
기능: 경기 시작 30분 전, 골 발생 시 알림

추가 데이터:
- UserPreferences: pushEnabled, matchAlertTime, goalAlertEnabled
- NotificationHistory: id, userId, type, content, readAt

UI 위치:
- 설정 페이지 (알림 on/off)
- 알림 히스토리

이유: 경기를 놓치지 않게 하고, 실시간 참여 유도!
```

#### 3. 📝 응원가 북마크/좋아요
```
기능: 자주 부르는 응원가 저장, 빠른 접근

추가 데이터:
- UserBookmarks: userId, songId, createdAt
- SongStats: id, bookmarkCount, viewCount

UI 위치:
- 응원가 상세 페이지 하단 "북마크" 버튼
- 마이페이지 > 북마크 목록
- 메인 페이지 "자주 부르는 응원가" 섹션

이유: 팬들이 자주 쓰는 응원가를 빠르게 찾을 수 있어요!
```

#### 4. 🎧 응원가 연습 모드
```
기능: 속도 조절, 반복 재생, 박자 강조

추가 기능:
- YouTube Player 제어 (속도 0.5x ~ 2x)
- 박자 하이라이트 (특정 가사 강조)
- A-B 반복 (구간 반복)
- 카라오케 모드 (가사 숨기기)

UI 위치:
- 응원가 상세 페이지 > "연습 모드" 버튼
- 플레이어 컨트롤 추가

이유: 초보 팬들도 쉽게 응원법을 익힐 수 있어요!
```

#### 5. 🏟️ 경기장 안내 & 도착 정보


## 🎨 UX 개선 제안

### 1. 홈 페이지 최적화
```
현재: 검색창 + 카테고리 + HOT 응원법

개선:
┌─────────────────────────┐
│  다가오는 경기 (카드)      │ ← 경기 일정
├─────────────────────────┤
│  검색창                  │
├─────────────────────────┤
│  상황별 응원 | 선수별 응원 │
├─────────────────────────┤
│  🔥 오늘의 Hot 응원법    │
│  📌 자주 부르는 응원가    │ ← 북마크 (로그인 시)
└─────────────────────────┘
```

### 2. 상세 페이지 개선
```
현재: 영상 + 참고사항 + 가사

개선:
┌─────────────────────────┐
│  YouTube 영상            │
│  [연습 모드] [북마크]      │ ← 버튼 추가
├─────────────────────────┤
│  📝 참고사항              │
├─────────────────────────┤
│  🎤 응원가 가사            │
│  [박자 강조] [속도 조절]   │ ← 연습 모드
├─────────────────────────┤
│  📎 관련 응원가 (추천)    │ ← 관련 항목
└─────────────────────────┘
```

### 3. 하단 네비게이션 개선
```
현재: 홈 | 상황별 | 선수별 | 검색

개선:
🏠 홈 | ⚽ 경기 | 🎤 응원가 | 👤 마이페이지

"경기" 탭: 일정, 실시간 스코어, 경기장 정보
"응원가" 탭: 상황별/선수별 통합, 검색
```
